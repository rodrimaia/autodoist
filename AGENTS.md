## Agent skills

### Issue tracker

Issues live in GitHub Issues for `rodrimaia/autodoist`; external PRs are also a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Use the default triage label vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

This is a single-context repo: use `CONTEXT.md` at the root and ADRs under `docs/adr/`. See `docs/agents/domain.md`.
